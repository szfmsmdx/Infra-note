# 二分开闭区间写法
区间写法分为：
- [l,r]
- [l,r)

### 闭区间二分
首先看闭区间，那么 l 是有可能等于 r 的，此时其实也需要进入循环，所以这个时候，while的写法应该是 l<=r 才对  
那么排除条件呢？我们知道 mid 是被搜过的哪一方，所以闭区间下一次是不会出现被搜索过的部分的，所以他应该是 +1、-1  
具体对应到写法上来看：
```cpp
int bin_search(vector<int>&nums, int target){
    int l = 0, r = nums.size() - 1;
    while(l <= r){
        int mid = (l + r) >> 1; 
        if(nums[mid] == target) return mid;
        else if(nums[mid] > target) r = mid - 1;    // mid被搜过所以更新为 mid-1
        else l = mid + 1;   // 同理更新为 mid+1
    }
    return -1;
}
```

这个时候其实要在循环里面判断有没有被搜过，否则的话，出循环就是l和r要差一位要特判一下nums[l]、nums[r]

### 开区间二分
对比来看写法：
```cpp
int bin_search(vector<int>&nums, int target){
    int l = 0, r = nums.size();
    while(l < r){
        int mid = (l + r) >> 1;
        if(nums[mid] == target) return mid;
        else if(nums[mid] < target) l = mid + 1;
        else r = mid;
    }
    return nums[l] == target ? l : -1;
}
```

那么这里由于是开区间，所以while的写法就是 l < r，因为r那半边是开的  
此外在更新的时候，如果nums[mid] > target的话，那么r是可以给到mid的，因为这半边是开的所以不会再搜mid  
最后出循环应该是 l=r 所以看一下这个位置是不是要找的即可  
左闭右开写法的一个好处是可能搜到的是 nums.size() 这个位置，也就是说，这对STL风格是特别合适的，因为这个位置正好可以插入下一个元素什么的

# lower bound、upper bound写法区别
首先还是搞清楚 lower bound 和 upper bound 的区别：
- lower bound：找到第一个 >= target 的位置
- upper bound：找到第一个 >  target 的位置

那么其实关键点在于哪怕 nums[mid] == target，mid 可能也不是答案  
所以应该分 nums[mid] >= target 来看，去收缩右边界找第一个 lower bound的位置  
如果看 nums[mid] <= target 的话，那么扩张右边界应该是找 upper bound的位置

所以写法如下：
#### 闭区间写两种bound
写 lower bound：
```cpp
int lower_bound(vector<int>&nums, int target){
    int l = 0, r = nums.size() - 1;
    int res = -1;
    while(l <= r){
        int mid = (l + r) >> 1;
        if(target <= nums[mid]){
            r = mid - 1;
            res = mid;
        }
        else l = mid + 1;
    }
    return res;
}
```
可以用一个 res 来记一下，因为出循环还是和上面说的，l 和 r 是错开1位的，要特判比较麻烦，所以在循环里面更新

写 upper bound：
```cpp
int upper_bound(vector<int>& nums, int target){
    int l = 0, r = nums.size() - 1;
    int res = -1;
    while(l <= r){
        int mid = (l + r) >> 1;
        if(nums[mid] > target){
            res = mid;
            r = mid - 1;
        }else l = mid + 1;
    }
    return res;
}
```

#### 开区间写两种bound
开区间的话，其实就没有太大变化：
写 lower bound：
```cpp
int lower_bound(vector<int>&nums, int target){
    int l = 0, r = nums.size();
    while(l < r){
        int mid = (l + r) >> 1;
        if(nums[mid] >= target) r = mid;
        else l = mid + 1;
    }
    return nums[l] == target ? l : -1;
}
```

写 upper bound：
```cpp
int upper_bound(vector<int>& nums, int target){
    int l = 0, r = nums.size();
    while(l < r){
        int mid = (l + r) >> 1;
        if(nums[mid] > target) r = mid;
        else l = mid + 1;
    }
    return l;
}
```

# 什么时候用二分？
在做题中，我们遇到什么情况或者是什么题型的时候该往二分上面去考虑呢？  
一般来说，当题目具有：
- 单调性
- 两段性
这些特性的时候，我们可以考虑往二分上面去靠

一些经典题型有：
- 显示的有序数组查找
- **答案具有二分性**（枚举答案！！！！）
  - 假设答案为 x 可行，那么比 x 大也可行，比 x 小不可行，就构成了一个：False、False、...、True、True的两段性答案，此时我们考虑使用二分去枚举答案
- 寻找峰顶或者拐点（局部是有序的），比如：
  - LC162
  - LC153
  - LC852
- 涉及平方根
  - 浮点数二分吧，其实也可以归到峰顶、低谷的类型里面去
